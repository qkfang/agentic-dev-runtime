# Control Plane Frontend

React application for viewing and managing scopes in the Agentic Dev Runtime.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Runs on port 3000 by default.

## Build

```bash
npm run build
```

## Environment Variables

- `VITE_API_URL` - Backend API URL (default: http://localhost:3001)
